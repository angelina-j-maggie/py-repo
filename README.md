# py-repo

A template structure for repos.

## Structure

```
  README.md

  dev
  |
  |-- contains code used in products

  doc
  |  
  |--templates
     |
     |-- contains template code files

  exploration
  |
  |-- contains code used in exploratory analysis

  LICENCE
```

## Usage

Clone this repo to provide a template structure for new projects.
